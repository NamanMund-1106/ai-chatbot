# AI-Powered Chatbot 🤖

An intelligent chatbot built with Python, NLTK, Flask, and SQLite for customer support and FAQs.

## Features
- Natural Language Processing using NLTK
- Pattern-based conversational responses
- REST API built with Flask
- Conversation logs saved in SQLite database
- Futuristic dark chat UI with typing animation
- Quick reply chips for common queries

## Tech Stack
- Python
- NLTK
- Flask
- SQLite

## How to Run
1. Clone the repo
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install dependencies: `pip install flask nltk`
5. Run: `python app.py`
6. Open browser at `http://127.0.0.1:5000`
